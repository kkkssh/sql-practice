# SQL Practice

This repository contains my SQL practice as I build a strong foundation in data analysis.

## Structure

This repository is organized to gradually build SQL skills, starting from basic queries and expanding into more advanced topics.

## Files

- basic_queries.sql  
  → Covers fundamental SQL concepts:
  - CREATE TABLE
  - INSERT
  - UPDATE
  - DELETE
  - SELECT
  - WHERE (basic & advanced filtering)
  - ORDER BY
  - DISTINCT
  - ALTER TABLE

## Dataset

The examples use a simple `students` table with fields such as:
- id
- name
- student_year
- gender
- major
- country

## How to Run

1. Open your SQL client (e.g., MySQL Workbench, DBeaver)
2. Run the `basic_queries.sql` file
3. The script will create a database and table, then execute example queries

## Goal

To develop practical SQL skills for data analysis and real-world problem solving.

## Roadmap

- JOIN queries
- GROUP BY and aggregation
- Real-world data analysis examples
